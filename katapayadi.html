Katapayadi Encoding System

The Katapayadi system assigns values to consonants for numeric encoding. Vowels and matras are ignored.

  • क ख ग घ ङ = 1, 2, 3, 4, 5
  • च छ ज झ ञ = 6, 7, 8, 9, 0
  • ट ठ ड ढ ण = 1, 2, 3, 4, 5
  • त थ द ध न = 6, 7, 8, 9, 0
  • प फ ब भ म = 1, 2, 3, 4, 5
  • य र ल व = 6, 7, 8, 9
  • श ष स ह = 6, 1, 2, 3
  • क्ष त्र ज्ञ = 6, 6, 8
